# Cursor Rules Documentation

This directory contains comprehensive documentation for the Jira Agile Metrics project, organized by purpose and topic.

## File Structure

- **`overview.md`** - Project overview, features, quick start, and high-level architecture
- **`architecture.md`** - Technical architecture, patterns, data structures, and implementation details
- **`development.md`** - Development workflow, code style guidelines, and common development tasks
- **`api.md`** - API reference documentation for all public interfaces
- **`patterns.md`** - Common code patterns, best practices, and reusable examples
- **`testing.md`** - Testing guidelines, patterns, and best practices

## Usage

These files are automatically loaded by Cursor to provide context when working on the project. They help maintain consistency and provide guidance for:

- Understanding the project structure and architecture
- Following coding standards and patterns
- Implementing new features correctly
- Writing tests effectively
- Debugging and troubleshooting
